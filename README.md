## 📌 Overview

TrackSure is a full-stack web application designed to improve rural service delivery by enabling users to:

* Book essential services (LPG, Water, Maintenance)
* Track service status in real-time
* Raise complaints
* Earn reward points
* View dashboard insights

---

## 🧩 Tech Stack

* **Backend:** Python (Flask)
* **Frontend:** HTML, CSS, JavaScript
* **Database:** SQLite
* **UI:** Dark Theme (Modern Responsive Design)

## ⚙️ Features

### 🔐 Authentication

* User registration
* Login system

### 📦 Service Booking

* LPG / Water / Maintenance
* Address + time selection

### 📍 Order Tracking

* Status updates:

  * Booked → Assigned → On the Way → Delivered

### ⚠️ AI Delay Alert

* Shows delay warning based on conditions

### 🧾 Complaint System

* Submit and track complaints

### 🎁 Rewards System

* Earn points for:

  * Booking services

### 🌐 Multi-language Support

* English & Tamil toggle
